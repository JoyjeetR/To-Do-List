## To-Do List Application

A simple **Command-Line To-Do List Application** built using Python.  
This program allows users to **add, view, and remove tasks**, with all tasks stored persistently in a CSV file so that data is preserved between sessions.

## Features

- Add a single task  
- Add multiple tasks at once  
- View all tasks  
- Remove a task using its serial number  
- Persistent storage using CSV  
- Simple and beginner-friendly CLI interface  

## Technologies Used

- **Python**
- **CSV Module (built-in)**

## Application Menu
When you run the program, the following menu appears:

1. Add
2. Remove
3. View
4. Exit

## Data Storage
All tasks are stored in ToDoList.csv
CSV format:
Serial Number, Task Description
Data remains saved even after exiting the program

## Future Enhancements
- Auto-generate serial numbers
- Add task completion status
